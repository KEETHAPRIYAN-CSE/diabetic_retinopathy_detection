# Sample images go here (optional)

Drop a few APTOS fundus images here (one per grade is a nice demo set, e.g.
`grade0_sample.png` ... `grade4_sample.png`) if you want the "Use a bundled
sample image" checkbox in the Streamlit app to have something to show during
a live demo without needing to upload a file on stage.

This is entirely optional — the app works fine with manual upload only.

You can grab a few images directly from the APTOS `train_images/` folder
after downloading the dataset in the Colab notebook (Section 2), or from
`https://www.kaggle.com/c/aptos2019-blindness-detection/data`.
